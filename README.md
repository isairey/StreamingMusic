## ✨ Características

- 🔍 **Búsqueda de canciones en línea**
- 🎧 **Streaming de música en tiempo real**
- ⬇️ **Descarga de canciones para modo offline**
- 🔥 **Calidad de audio hasta 320kbps (MP3 / M4A)**
- 🏷️ **Metadatos ID3 automáticos**
- 📝 **Soporte para letras sincronizadas**
- ⚡ **Interfaz rápida, ligera y moderna**
